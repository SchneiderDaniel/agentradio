# Cosk

Codebase-skeleton orchestration scaffold for Cosk v2.

## Packages
- `extraction/` — tree-sitter parsing and pruning
- `indexing/` — embeddings and LanceDB indexing
- `graph/` — NetworkX graph operations
- `mcp/` — MCP server and tool surface
- `safety/` — traversal and cycle guardrails

## Setup
```bash
python -m pip install -e .
```

## Status
Initial scaffold only; implementation arrives in later issues.
